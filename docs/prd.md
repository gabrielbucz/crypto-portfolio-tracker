## 📌 Descrição

O sistema tem como objetivo permitir que usuários acompanhem o preço de criptomoedas em tempo real e gerenciem um portfólio pessoal de investimentos.

A aplicação resolve o problema de acompanhar manualmente preços e calcular lucros/prejuízos, centralizando essas informações em uma interface simples e responsiva.

---

## 👥 Atores do Sistema

* Visitante (usuário não autenticado)

---

## 📖 Histórias de Usuário

### 🔹 Visualização de mercado

* Como usuário, quero visualizar uma lista de criptomoedas com seus preços atualizados para acompanhar o mercado.
* Como usuário, quero ver a variação de preço (positiva ou negativa) para entender tendências.

### 🔹 Favoritos

* Como usuário, quero adicionar criptomoedas aos favoritos para acompanhar mais facilmente.
* Como usuário, quero remover moedas dos favoritos para manter minha lista organizada.
* Como usuário, quero visualizar minha lista de favoritos em uma página dedicada.

### 🔹 Portfólio

* Como usuário, quero cadastrar minhas criptomoedas informando quantidade e preço de compra para controlar meus investimentos.
* Como usuário, quero visualizar meu portfólio em forma de tabela para analisar meus ativos.
* Como usuário, quero ver o lucro ou prejuízo automaticamente calculado para cada moeda.

### 🔹 Formulários

* Como usuário, quero preencher um formulário com validação para garantir que os dados inseridos estejam corretos.
* Como usuário, quero receber mensagens de erro ao preencher dados inválidos.

### 🔹 Responsividade

* Como usuário, quero acessar o sistema pelo celular ou computador sem perder usabilidade.
